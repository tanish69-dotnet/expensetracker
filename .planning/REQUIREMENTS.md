# Requirements: Expanse

**Defined:** 2026-04-11
**Core Value:** Low-friction, aesthetic expense tracking that makes financial awareness effortless.

## v1 Requirements

### Expense Operations

- [ ] **EXP-01**: User can add an expense with Amount, Category, and Date.
- [ ] **EXP-02**: User can view a scrollable list of all recorded expenses.
- [ ] **EXP-03**: User can delete any expense from the list.
- [ ] **EXP-04**: Persistent storage so data remains after refresh.

### Dashboard & Metrics

- [ ] **DASH-01**: Dashboard shows the total spending across all expenses.
- [ ] **DASH-02**: Categories are clearly labeled (Food, Travel, Bills, etc.).

### UI/UX (Premium Feel)

- [ ] **UI-01**: Dark Theme by default.
- [ ] **UI-02**: Glassmorphism cards for expense items.
- [ ] **UI-03**: Responsive design (Mobile & Desktop).

## v2 Requirements (Deferred)

- **AUTH-01**: User login and cloud sync.
- **CHRT-01**: Pie charts for category distribution.
- **EDIT-01**: Edit existing expenses.

## Out of Scope

| Feature | Reason |
|---------|--------|
| Bank Auto-Sync | High complexity, security concerns for v1. |
| Multi-Currency | Defer to later milestone. |
